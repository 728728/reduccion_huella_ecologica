1. Determina los fundamentos teóricos y metodológicos de las páginas web.


* Una página web estática es un sitio de Internet que muestra el mismo contenido para todos los usuarios, en vez de proporcionar contenido personalizado a la medida de cada usuario, y que no se actualiza a menudo.

* El desarrollo inicial de una página web estática es un proceso más simple ya que no requiere sofisticadas habilidades de programación. Las páginas web estáticas solamente requieren la comprensión de un lenguaje de programación básico como HTML y pueden ser creadas por diseñadores de sitios de forma relativamente barata.

* La relativa simplicidad de la fase de desarrollo ocasiona que las páginas web estáticas sean atractivas para los usuarios que tienen en mente mantener los costos iniciales a un nivel bajo.
* Las páginas web estáticas a menudo se usan para sitios orientados hacia la prestación de ofertas o información estable a los visitantes (información que probablemente no sea necesario actualizar o alterar durante un largo periodo de tiempo).

2. Diagnosticar el estado actual del uso de las páginas web estáticas.


Recapitulando, en el punto anterior declaramos que una página web estática es ideal para aquellas empresas que sólo buscan la transmisión de información, como por ejemplo, la publicación e información de un artículo y/o producto. Ya que, una página estática es una página orientada a la difusión de información que no sea cambiada en mucho tiempo.

Pero esto no significa que no cuente con desventajas:

* Las páginas web estáticas tienen muchas desventajas para los propietarios de sitios que quieran crear una atmósfera o apariencia de interactividad con los visitantes, ya que no existen opciones para crear una atmósfera interactiva en una página estática con contenido personalizado para usuarios individuales.

*Las páginas web estáticas tampoco pueden acceder a la información del usuario de la forma en la que las páginas dinámicas pueden hacerlo, lo que significa que los propietarios de dichos sitios pierden la oportunidad de saber más acerca de las personas que visiten el sitio.

* Para actualizar una página estática se requiere que el usuario conozca el lenguaje de programación relevante y que sea capaz de hacer cambios usando el código incrustado en la página web. El propietario de una página web estática necesita conocer la programación de su página para actualizarla, o de lo contrario tendría qué pagar para que un diseñador lo haga.

HOST: Un host o anfitrión es un ordenador que funciona como el punto de inicio y final de las transferencias de datos. Más comúnmente descrito como el lugar donde reside un sitio web. Un host de Internet tiene una dirección de Internet única (dirección IP) y un nombre de dominio único o nombre de host. El término host también se utiliza para referirse a una compañía que ofrece servicios de alojamiento para sitios web. Hay host de paga, como: Webempresa, HostGator o Bana Hosting (entre otros). Hay host barato, como: IPage o HostGator (entre otros). Y hay host gratuito, como: Hostinger (entre otros). Cada uno cuenta con características en soporte, actualizaciones, herramientas y velocidad diferente. Por lo que, dependerá mucho el número de las visualizaciones que generes por mes.

Diseñadores web: ¿Cómo cobran e implementan su trabajo?

Un diseñador web determina el precio de su trabajo como en cualquier otro servicio profesional lo determina; mediante la oferta y la demanda del cliente, aunque siempre existen elementos que debes tener en cuenta antes de pedir presupuesto o contratar estos servicios de diseño de páginas Web. Existen varios determinantes que hacen que el diseñador web fije su precio. Encontré que una diseñadora fija su precio en el cálculo del precio de una página/hora conforme a sus ingresos mensuales. Por lo que, en conclusión, cualquier diseñador web diseñará e implementará su producto según lo que el cliente le pida. Haciendo así, un presupuesto base basándose en el tiempo de la elaboración, los recursos requeridos y la complejidad y/o simplicidad del proyecto.

Proceso de publicación/instalación de una página web:

* Crear o modificar todos los archivos que van formar su sitio web (página de inicio, imágenes, archivos de descarga, etc.) en su ordenador de forma local.

* Terminadas las modificaciones debemos colocar todos los archivos que forman el sitio web en el servidor/host. Este se encargará de ubicarlo y proponerle un sitio en la web (dirección de URL).

4. Validar la efectividad de la página web estática que apoye a la transmisión de información para la reducción de la huella ecológica.


Entendemos por contenido web todas aquellas producciones que alimentan un sitio web y que pueden ser leídas, visualizadas o escuchadas por el visitante como parte de su experiencia de usuario. Por lo que debes aprender que medir los resultados de todo aquello que realices en Internet tiene que ser, a partir de este instante, una prioridad absoluta. Porque solo el análisis nos permite la optimización y sólo la optimización nos llevará a la conversión.

Para averiguar si nuestro contenido web está generando los resultados que esperamos, tenemos una serie de indicadores esenciales:


* Tráfico Web: El tráfico web hace referencia al número de usuarios que ha visitado la página donde se encuentra tu contenido. Esto nos da una idea de la visibilidad que ha obtenido y de su alcance. Para averiguar esta cifra, nuestro aliado es Google Analytics.

* Tiempo de Permanencia en la página: El tiempo de permanencia nos indica durante cuánto tiempo han interactuado nuestros usuarios con nuestro contenido.
Por lo general, cuanto mayor tiempo de permanencia se registre nuestro contenido, mayor interés se deduce que ha tenido el usuario por él. Google Analytics es la herramienta ideal para visualizarlo.
* Social Media Sharing: El grado de viralidad que obtenga tu contenido será un indicador del valor que le asignan tus usuarios. Si tu público considera que tu contenido es útil lo compartirá con el resto de sus contactos. Puedes usar la herramienta WordPress o Shareaholic para verlo.

* Feedback de los clientes: Si tu contenido obtiene muchos comentarios significará que tu contenido llama a la acción y consigue captar a tu público.
Sin embargo, recuerda que no todo está en la cantidad, también debes considerar la calidad de ese feedback. Podemos obtener 100 comentarios en una de nuestras páginas de contenido, pero si todos ellos son negativos, evidentemente algo no funciona.
Lo mismo sucede con el inbound links, que son el número de ‘me gusta’ que tiene tu página.
