2. ¿CÓMO DIAGNOSTICAR EL ESTADO ACTUAL DEL USO DE LAS PÁGINAS WEB ESTÁTICAS?


Las páginas web estáticas son ideales para transmitir y difundir información entorno a una empresa (Quienes son, Misión, Visión, etc.) aunque cuentan con desventajas al momento de la interacción con un cliente. Por ejemplo, no pueden mostrar o negar cierto contenido a determinados usuarios. Mas sin embargo, esa restricción, también hace que la página sea más rápida puesto que le muestra la misma información a todos.

Como la página está orientada a la difusión de información, una página web estática es el sitio ideal (tanto en costo como en funciones) para crearla. Además, de que, el código fuente no será modificado tan a menudo, por lo que el contratar a un diseñador para que se encargue de actualizarla o cambiarla, no será necesario.
